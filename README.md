## Description

Developed on [Nest](https://github.com/nestjs/nest) framework TypeScript.

## Installation

```bash
$ yarn install
```

## Migration & seeding using Prisma

```bash
$ prisma migrate dev --name init
$ npx prisma db seed
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
